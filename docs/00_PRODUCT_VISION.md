# Visão do produto

O sistema reduz variabilidade e perda de conhecimento na produção cervejeira. O diferencial não é apenas registrar dados: é comparar plano e execução, recomendar correções determinísticas, rastrear cada insumo e aprender com lotes encerrados.

## Personas

Administrador, cervejeiro, operador de produção, responsável por qualidade, estoque/compras, auditor e consulta.

## Jornadas críticas

1. Receita → versão publicada → OP → reserva de insumos.
2. Água → mostura → fervura → resfriamento → transferência.
3. Fermentação → maturação → envase → lote de embalagem.
4. Entrada de insumo → consumo → lote → embalagem → recall.
5. Limpeza → verificação → sanitização → liberação do equipamento.

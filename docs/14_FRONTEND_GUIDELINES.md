# Frontend Angular

Angular 22 com estrutura feature-first: `core`, `shared`, `features/<module>`. Rotas lazy, componentes standalone, Signals, zoneless, formulários tipados/Signal Forms e serviços de API compatíveis com OpenAPI. RxJS continua responsável por fluxos assíncronos; store global não é padrão.

Telas de produção priorizam tablet/celular, alto contraste, botões grandes, cronômetros persistentes e retomada. Estados loading/empty/error/disabled são obrigatórios. Acessibilidade: teclado, foco, rótulo, contraste e alertas que não dependem apenas de cor.

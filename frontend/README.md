# Frontend de referência

Gerar a aplicação com Angular CLI 22, standalone, routing e SCSS usando Node 24 LTS. Commitar `package-lock.json` e executar CI com `npm ci`. Use Vitest, que é o runner padrão para novos projetos Angular, e Playwright para E2E.

Copie a organização abaixo após o CLI criar os arquivos de workspace. Não introduza NgRx, Tailwind ou PWA na Sprint 00 sem uma história que justifique; Angular Material/CDK e tokens CSS formam a base inicial.

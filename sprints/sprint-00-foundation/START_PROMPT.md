# Prompt para iniciar a Sprint 00

Copie e envie o texto abaixo à IA responsável pelo desenvolvimento:

> Execute integralmente a Sprint 00 deste kit, seguindo `sprints/sprint-00-foundation/AI_EXECUTION.md` e `RUNBOOK.md`. Comece pelo preflight e pela criação ou vinculação segura do repositório remoto; gere os projetos backend e frontend reais; configure ambiente, banco, migrations, contratos, observabilidade, testes e CI; valide tudo em checkout limpo; faça os commits e pushes previstos; e só encerre quando `ACCEPTANCE.md` tiver evidências reais. Se faltarem provedor, proprietário, nome ou visibilidade do repositório, faça uma única pergunta reunindo esses campos. Não use force-push, não sobrescreva trabalho existente, não versione segredos e não implemente histórias da Sprint 01.

A solicitação para executar esta sprint autoriza alterações locais, criação dos projetos, commits e pushes normais no repositório confirmado. Não autoriza apagar recursos, reescrever histórico, alterar visibilidade, criar serviço pago, publicar release ou modificar outro repositório.

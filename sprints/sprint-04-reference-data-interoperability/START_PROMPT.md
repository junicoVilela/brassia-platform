# Iniciar a Sprint 04

Use este prompt no repositório real da BrassIA:

> Execute integralmente a Sprint 04 deste kit. Considere as Sprints 00–03 já desenvolvidas e inspecione o código antes de propor alterações. Preserve contratos e dados existentes, use migrations aditivas e trabalhe história por história. Comece comparando a implementação real de REC-006 com REC-007/REC-008 para evitar duplicação. Pare apenas se faltar uma decisão de licença, acesso ou produto que altere materialmente a solução; caso contrário, continue até os critérios de aceite, testes, documentação e evidências estarem concluídos.

## Pré-requisitos

- checkout do repositório real;
- build e testes das Sprints 00–03 verdes;
- banco/migrations existentes acessíveis;
- decisão sobre quais datasets podem ser publicados;
- exemplos de arquivos BeerJSON e BeerXML permitidos para teste.

## Resultado esperado

Uma versão executável com dados de referência versionados, importação/exportação segura, calculadoras ampliadas e comparação guiada, sem regressão no que já foi desenvolvido.
